ETIQUETAS DE PARRAFO

esta la etiqueta p que es la que se usa para hacer parrafos

tambien esta la pre, que tiene unas caracteristicas predefinidas: 

        sirve mucho para poemas

        ya que toma los saltos de linea

        de forma automatica

tenemos los saltos de
linea con la etiqueta br

y una linea para definir una seccion o un texto de otro con la etiqueta hr
